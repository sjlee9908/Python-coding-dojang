클래스 속성과 정적, 클래스 메서드

클래스속성
class 클래스이름:
	속성=값
클래스 속성은 클래스에 속해있으며 모든 인스턴스에 속함

클래스 속성 - 모든 인스턴스가 공유, 인스턴스 전체가 사용해야 하는 값을 저장할 때 사용
인스턴스 속성 - 인스턴스 별로 독립, 각 인스턴스가 값을 따로 저장할 때 사용

비공개 클래스속성
class 클래스이름:
	__속성 = 값

정적메서드
class 클래스이름:
	@staticmethod
	def 매서드(매개변수1, 매개변수2):
		코드
매개변수에 self 지정 x
메서드의 실행이 외부상태에 영향을 미치지 않는 순수함수를 만들 때 사용

클래스메서드
class 클래스이름:
	@classmethod
	def 메서드(cls, 매개변수1, 매개변수2)
cls로 클래스 속성에 접근

클래스 속성에 접근할 필요가 있는 경우엔 클래스메서드 사용
클래스 속성에 접근할 필요가 없는 경우엔 정적메서드 사용

if. else를 사용하면 조건이 만족할 때. 만족하지 않을 때각각 다른 코드 실행 가능

else는 if 조건문 뒤에 오며 단독으로 사용 불가

조건식이 참 --> if
조건식이 거짓 --> else

조건부표현식
>>> x=5
>>> y=x if x==10 else 0

들여쓰기 유의

참: True, 0이 아닌 숫자(진수상관x), 빈 문자열이 아닌 문자열 리스트 튜플 딕셔너리 세트
거짓: False, 0, 빈문자열 리스트 튜플 딕셔너리 세트, None

조건식 여러개 지정 --> and, or 사용
and, or를 사용해 중첩 if문을 줄일 수 있음

문자열 응용

문자열바꾸기
.replace('바꿀문자열', '새문자열')

문자바꾸기
str.maketrans('바꿀문자','새문자')
.translate(테이블)

문자열 --> 리스트
.split('기준문자열')

리스트 --> 문자열
'요소사이에들어갈문자열'.join(리스트)

소문자 --> 대문자
.upper()

대문자 --> 소문자
.lower()

왼쪽문자삭제
.lstip('삭제할 문자')

오른쪽문자삭제
.rstrip('삭제할 문자')

양쪽문자 삭제
.strip('삭제할 문자')

문자열 왼쪽 정렬
.ljust(숫자)

문자열 오른쪽 정렬
.rjust(숫자)

문자열 가운데 정렬
.center(숫자)

메서드 체이닝
여러개의 메서드를 줄줄이 연결하는 것
ex) .rjust().upper()

문자열 왼쪽에 0 채우기
.zfill(숫자)

문자열 위치 찾기
.find('찾을문자열')	or	.index()

오른쪽에서부터 문자열 찾기
rfind('찾을문자열')	or	.rindex()

문자열 개수 세기
.count('찾을 문자열')

문자열서식지정자
'%s'%'문자열'

숫자서식지정자
'%d'%숫자

소수서식지정자
'%f'%소수

서식지정자로 문자열 오른쪽 정렬
'%길이s'%'문자열'

서식지정자로 문자열 왼쪽 정렬
'%-길이s'%'문자열'

서식지정자로 값 여러개 넣기

서식지정자로 문자열 오른쪽 정렬
'%d %s'%(숫자, '문자열')

format 메서드 사용
'{인덱스}'.format(값)

format 메서드로 값을 여러개 넣기
'{인덱스}{인덱스}{인덱스}'.format(값, 값, 값)
인덱스 생략시 format의 값 순서대로 들어감
인덱스 대신에 name 지정 가능, 단 format에서 name에 값을 할당해주어야함

문자열 포메팅에 변수를 그대로 사용하기
f'문자열{변수}'

format 메서드로 문자열 왼쪽 정렬
'{인덱스:<길이}'.format('문자열')

format 메서드로 문자열 오른쪽 정렬
'{인덱스:>길이}'.format('문자열')

숫자개수 맞추기
'%0개수d'%숫자
'{인덱스:0개수d}'.format(숫자)

딕셔너리 응용

.setdefault(키, 값)
키-쌍 추가

.update(키=값)
기존에 있던 키-값 수정, 추가
키가 문자열일때만 사용가능, 키가 문자열이 아닐땐 키=값대신 딕셔너리를 넣음

.pop(키)
특정 키-값 삭제 후, 값 반환
=del 딕셔너리[키]

.popitem()
마지막 키-값 삭제, 삭제한 키-쌍 튜플로 반환

.clear()
딕셔너리의 모든 키-값 삭제

.get(키)
딕셔너리의 특정 키의 값을 가져옴

.items()
딕셔너리의 모든 키-값을 가져옴

.keys()
딕셔너리의 모든 키를 가져옴

.values()
딕셔너리의 모든 값을 가져옴

dict.fromkeys(키리스트)
키 리스트로 딕셔너리의 키를 생성

딕셔너리 표현식
{키: 값 for 키, 값 in 딕셔너리}

딕셔너리 표현식에 if문 사용
{키:값 for 키, 값 in 딕셔너리 if 조건식}

딕셔너리 안에서 딕셔너리 사용 가능

딕셔너리 할당, 복사
할당 =
복사 .copy
